if you are a teacher : (ONLY ONE TIME) then create an username (bottom-right) for all your students then tell the student his username. Be careful the uppercases in the username are meaningful so please give the exact username to the according user.

Classses


Class that happened

To add a class which happened : example this saturday 1) select that date on the calandar 2) then put a check next to the checkbox class happened 3) press save very important

To remove a class which did not happen : example you pressed the wrong date 1) select that date on the calandar 2) then remove the check next to the checkbox class happened 3) press save very important

Class that were missed

To add a class which was missed : 1) select that date on the calandar 2) then put a check next to the checkbox class missed 3) press save very important

To remove a class which did not happen : example you pressed the wrong date 1) select that date on the calandar 2) then remove the check next to the checkbox class missed 3) press save very important
